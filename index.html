Dartmouth Timetable Scanner

This is a project I worked at the end of winterim to automate the process of searching for empty seats during the Dartmouth Add/Drop period. The program scans the online timetable, and texts the user when a spot opens up. Click here to see the GitHub repo!

CAN Module

This is my most recent project at Dartmouth Formula Racing. It translates analog and digital sensor readings into messages that can be transmitted over the industry standard CAN protocol. Click here to see the GitHub repo!
